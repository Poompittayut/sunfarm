# คู่มือเชื่อม HTML กับ Google Sheet (อ่านสด)

ทำครั้งเดียว ~10 นาที · หลังจากนี้แก้ข้อมูลใน Sheet แล้วกดรีเฟรชใน HTML ได้เลย

```
แก้ข้อมูลใน Google Sheet  ──(Apps Script ส่ง JSON)──►  SunFarm_CoopModel.html อ่านสด
```

---

## ขั้นตอน

### 1) สร้าง Google Sheet ใหม่
- ไปที่ https://sheets.new (จะได้ชีทเปล่า 1 อัน ชื่อ Sheet1 — ไม่ต้องแก้)
- ตั้งชื่อไฟล์อะไรก็ได้ เช่น `Sunfarm ผสมพันธุ์ไก่`

### 2) เปิด Apps Script แล้ววางโค้ด
- เมนู **Extensions → Apps Script**
- ลบโค้ด `function myFunction(){}` ที่มีอยู่ทิ้ง
- เปิดไฟล์ **`SunFarm_AppsScript.gs`** (ในโฟลเดอร์นี้) คัดลอก **ทั้งหมด** มาวาง
- กด 💾 (Save)

### 3) รันสร้างตาราง (seed)
- ที่แถบบนสุด เลือกฟังก์ชัน **`seedData`**
- กด **▶ Run**
- ครั้งแรกจะขออนุญาตสิทธิ์ → Review permissions → เลือกบัญชี → Advanced → "Go to (project) (unsafe)" → Allow
  *(ปลอดภัย เป็นสคริปต์ของคุณเองที่เขียนลงชีทตัวเอง)*
- กลับไปดูที่ Google Sheet จะมี **12 แท็บ** ถูกสร้างพร้อมข้อมูลตั้งต้นครบ

### 4) Deploy เป็น Web app
- เมนู **Deploy → New deployment**
- กดรูปเฟือง ⚙ ข้าง "Select type" → เลือก **Web app**
- ตั้งค่า:
  - **Description:** อะไรก็ได้
  - **Execute as:** `Me`
  - **Who has access:** `Anyone`  ← สำคัญ (ต้องเป็น Anyone ไม่งั้น HTML ดึงไม่ได้)
- กด **Deploy** → คัดลอก **Web app URL**
  (หน้าตาแบบ `https://script.google.com/macros/s/AKfyc..../exec`)

### 5) วาง URL ลงใน HTML
- เปิดไฟล์ **`SunFarm_CoopModel.html`** ด้วย editor
- หาบรรทัด (อยู่ช่วงท้ายไฟล์ ในส่วน `<script>`):
  ```js
  const SHEET_API_URL = "";
  ```
- วาง URL ลงในเครื่องหมายคำพูด:
  ```js
  const SHEET_API_URL = "https://script.google.com/macros/s/AKfyc..../exec";
  ```
- เซฟ แล้วเปิดไฟล์ HTML ในเบราว์เซอร์

✅ เสร็จ! มุมขวาล่างจะมีไฟเขียว "ซิงค์จาก Google Sheet แล้ว"

---

## การใช้งานประจำวัน
- **แก้ข้อมูล:** แก้ในแท็บของ Google Sheet ได้เลย (เช่น แท็บ `แผนทดลอง`)
- **ดูผลใน HTML:** กดปุ่ม **↻ รีเฟรช** มุมขวาล่าง (หรือรีโหลดหน้า)
- ไม่ต้อง deploy ใหม่ — แก้ Sheet แล้วรีเฟรชได้เลย

> Deploy ใหม่เฉพาะตอนที่ **แก้โค้ด `.gs`** เท่านั้น (Deploy → Manage deployments → ✏ → Version: New version)

## ตารางใน Sheet ↔ แท็บใน HTML

| แท็บ Sheet | ผูกกับ | แก้อะไรได้ |
|-----------|--------|-----------|
| `แผนทดลอง` | 🗺️ แผนทดลอง + 🔀 เมทริกซ์ | คู่ผสม พ่อ/แม่/ลูก, จำนวน, วันฟัก (คอลัมน์ `batches` คั่นวันที่ด้วย ` \| `) |
| `ผังเล้า` + `ผังเล้ารวม` | 🏠 ผังเล้า | กรง: จำนวน/สี (hex) ต่อกรง · ยอดรวมต่อแถวอยู่ที่ `ผังเล้ารวม` |
| `Lockbook` | 📒 Lockbook | ทะเบียนไก่รายตัว (1 แถว = 1 ตัว) |
| `ไข่วันที่` + `ไข่รายวัน` | 🥚 ไข่รายวัน | วันที่ (คอลัมน์เดียว) + จำนวนไข่รายวัน (คอลัมน์ `daily` คั่นด้วย `,` ช่องว่าง=ไม่มีข้อมูล) |
| `onhand` | 🐤 ตู้ฟัก/ลูกไก่ | ผลฟักรายรอบ (%ฟัก/%เชื้อ คำนวณจากตัวเลขเหล่านี้) |
| `สีพันธุ์`/`สีรุ่น`/`META`/`สีแท็ก`/`สีsection` | สี/ป้ายกำกับ | ตารางอ้างอิงสี+ชื่อพันธุ์ |

### กฎสำคัญเวลาแก้ Sheet
1. **ห้ามเปลี่ยนชื่อหัวคอลัมน์** (แถวแรกของแต่ละแท็บ) — โค้ดอ่านตามชื่อหัว
2. **ห้ามเปลี่ยนชื่อแท็บ**
3. เพิ่ม/ลบ **แถว** ได้อิสระ (เว้นแถวว่างได้ ระบบข้ามให้)
4. วันที่พิมพ์เป็นข้อความได้เลย (ทุกช่องตั้งเป็น format ข้อความไว้แล้ว กันวันเพี้ยน)

## แก้ปัญหาเบื้องต้น
| อาการ | สาเหตุ/วิธีแก้ |
|-------|----------------|
| ไฟแดง "เชื่อมต่อไม่ได้" | เช็ก URL ถูกไหม · Who has access ต้องเป็น **Anyone** |
| เปิด URL แล้วเจอหน้า login Google | ตอน Deploy ตั้ง access ไม่เป็น Anyone → Deploy ใหม่ |
| แก้ Sheet แล้วไม่อัปเดต | กดปุ่ม ↻ รีเฟรช / รีโหลดหน้า |
| ข้อมูลเพี้ยนหลังแก้ | เช็กว่าไม่ได้เปลี่ยนชื่อหัวคอลัมน์/ชื่อแท็บ |
| อยากรีเซ็ตข้อมูลกลับค่าตั้งต้น | รัน `seedData` ใหม่ (จะเขียนทับทุกแท็บ) |
