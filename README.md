# 🐔 Sunfarm — ระบบจัดการการผสมสายพันธุ์ไก่

ระบบติดตามการทดลองผสมสายพันธุ์ไก่ในฟาร์มทดลอง — กรอกข้อมูลภาคสนาม → เก็บใน Google Sheet → แสดงผลภาพรวมแบบโต้ตอบได้

```
                       ┌─ [คนดู] เปิดดูได้เลย ไม่ต้องล็อกอิน (ปุ่มแก้ไขถูกซ่อน)
Google Sheet ──อ่าน──► CoopModel.html ┤
 (ฐานข้อมูล)   ▲       └─ [คนแก้ไข] กด 🔑 เข้าสู่ระบบ → แก้ไขได้ทุกอย่างในหน้าเดียว
               │
               └──เขียน── Form.html (ฟอร์มกรอกภาคสนาม/มือถือ) + CoopModel (โหมด editor)
```

## ไฟล์ในโปรเจกต์

| ไฟล์ | คืออะไร |
|------|---------|
| **`index.html`** | หน้า landing (GitHub Pages เปิดหน้าแรก) — 3 ปุ่ม: ฟอร์ม / ภาพรวม / ผังเครือญาติ |
| **`SunFarm_CoopModel.html`** | **หน้าหลัก** — ดูภาพรวม + แก้ไขครบในหน้าเดียว (แผนทดลอง · ผังเล้า · Lockbook · ไข่รายวัน · ตู้ฟัก · เมทริกซ์ผสม · จัดการสายพันธุ์) · **คนดูเห็นอย่างเดียว · login เพื่อแก้ไข** |
| **`SunFarm_Form.html`** | หน้าฟอร์มกรอกข้อมูล (มือถือได้) — ไข่รายวัน · ผลฟัก · เพิ่มไก่/Family |
| **`SunFarm_FamilyTree.html`** | ผังเครือญาติการผสมพันธุ์ (root tree) — ดูง่ายสำหรับคนทั่วไป |
| **`breeds.js`** | ตารางสายพันธุ์กลาง (สี/ชื่อ) ทุกหน้าโหลดร่วมกัน — **ต้อง deploy คู่ HTML เสมอ** |
| **`SunFarm_AppsScript.gs`** | Google Apps Script — seed ข้อมูล + เสิร์ฟ JSON (อ่านเปิด) + รับ action เขียนกลับ (ต้องมีรหัส) |
| **`SETUP_GoogleSheet.md`** | คู่มือ deploy Apps Script + ผูกกับ Sheet ทีละขั้น |
| **`handoff.md`** | เอกสารส่งต่องาน (โครงสร้างข้อมูล/สถาปัตยกรรมแบบละเอียด) |

> ไฟล์ข้อมูลต้นฉบับ `SB Tag (1).xlsx` **ไม่ได้อยู่ใน repo** (ข้อมูลจริงอยู่ใน Google Sheet)

## สิทธิ์การใช้งาน (login)

- **คนดู:** เปิดเข้ามาเห็นข้อมูลได้ทันที ไม่ต้องใส่รหัส — ปุ่มแก้ไขทั้งหมดถูกซ่อน
- **คนแก้ไข:** กด **🔑 เข้าสู่ระบบ** (มุมขวาบน) ใส่รหัส → แก้ไข/เพิ่ม/ลบ ได้ทุกอย่างจบในหน้าเดียว (สายพันธุ์ · คู่ผสม · ไก่/family · ไข่ · ผลฟัก · กรงผังเล้า)
- รหัสเก็บใน Apps Script (`WRITE_KEY`) เท่านั้น — **ไม่ commit รหัสจริงขึ้น repo** (โค้ดใน repo เป็น placeholder)

## โครงสร้างข้อมูล 5 ชั้น

```
ชั้น 0 แผนทดลอง  → พิมพ์เขียว: คู่ผสม พ่อ×แม่→ลูก + จำนวน family + ตารางฟัก
ชั้น 1 ผังเล้า    → จัดกรงจริง (ไก่พันธุ์/ไก่รุ่น)
ชั้น 2 Lockbook   → ทะเบียนไก่รายตัว จัดเป็น Family
ชั้น 3 ไข่รายวัน  → บันทึกการผสม + จำนวนไข่รายวัน
ชั้น 4 ผลฟัก      → ไข่เข้าฟัก→ลูกไก่ + KPI %ฟัก/%เชื้อ/%ตายโคม
```

เชื่อมกันด้วยรหัสไก่ เช่น `SB-26-G8-F01-F-001` = ไลน์–ปีเกิด–รุ่นพันธุกรรม–family–เพศ–ลำดับตัว

## เทคนิค

- HTML เป็น **single-file** (HTML+CSS+JS ในไฟล์เดียว) เปิดตรงๆ ได้ ไม่ต้องมี server
- เชื่อม Google Sheet ผ่าน **Apps Script Web App + JSONP** (เลี่ยง CORS, เปิดจาก `file://` ได้)
- ตั้งค่า URL ที่ตัวแปร `SHEET_API_URL` ในไฟล์ HTML

## เริ่มใช้งาน

ดู [`SETUP_GoogleSheet.md`](SETUP_GoogleSheet.md) — สรุป: สร้าง Sheet → วางโค้ด `.gs` → รัน `seedData` → Deploy เป็น Web app → วาง URL ลง HTML
